# 🦒 Marcel

A self-adapting agent build on top of claude-code that can rewrite itself.

## Goal

This project is meant to test the limits of vibe-coding by seeing if we can build a light agent-framework on top of claude-code. By building on top of claude-code the agent has access to it's own code-base and thus has the ability to rewrite an redeploy itself.

Secondary the idea for this project initiated with creating an agent that assist families in their day to day organization:
- tracking family events
- setting group reminders
- help organize day planning
- give reminders for activities
- ...

By giving the agent the ability to rewrite itself non-developer users in the family can more easily adapt Marcel to their needs.
